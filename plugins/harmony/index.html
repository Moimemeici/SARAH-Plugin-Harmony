Harmony

Plugin Logitech Harmony pour S.A.R.A.H. Contrôlez votre hub Harmony depuis SARAH.

Les sources sont disponibles sur GitHub: https://github.com/hobbe/SARAH-Plugin-Harmony

Installation

Après avoir installé le plugin à partir du store, il est nécessaire d'installer les dépendances Node.js. Exécuter le script de commandes install_plugin.bat .

Configuration

Vous devez configurer ce plugin en entrant vos identifiants Logitech Harmony :

  • login : votre identifiant (adresse email)
  • password : votre mot de passe
  • ip : l'adresse IP de votre hub Harmony

Commandes disponibles

  • Sarah, quel est l'état de Harmony ? : récupère l'état actuel du hub (allumé ou éteint)
  • Sarah, éteint Harmony : arrête l'activité courante
  • Sarah, démarre l'activité Télévision : démarre une activité, comme si vous cliquiez sur une activité de la télécommande
  • Sarah, arrête l'activité Télévision : arrête l'activité courante

Notes

Il sera nécessaire de modifier le fichier harmony.xml pour employer les noms des activités correspondantes à votre propre configuration de télécommande Harmony.

Pour l'instant, seules les fonctions de démarrage et d'arrêt d'activité sont implémentées. Avec Freebox et XBMC, je n'ai pas besoin des autres commandes de la télécommande Harmony.
